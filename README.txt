Prediction of three datasets with fully-connected NN, CNN, and RNN


Fully-connected NN: regression from the Boston Housing Dataset

Source dataset: https://www.cs.toronto.edu/~delve/data/boston/bostonDetail.html


CNN: multi-labe classification from the cifar-10 dataset

Source  dataset: https://www.cs.toronto.edu/~kriz/cifar.html


RNN: 2-day temperature forecast in Melbourne

Source dataset: https://raw.githubusercontent.com/jbrownlee/Datasets/master/daily-min-temperatures.csv'


Language: Python
